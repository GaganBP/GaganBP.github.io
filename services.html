Services I Offer

Excel Dashboards

I create professional Excel dashboards using advanced formulas, pivot tables, conditional formatting, charts, and slicers. Ideal for tracking KPIs, business metrics, or custom reports based on your data.

Power BI Reports

I design interactive and visually compelling reports in Power BI using DAX and Power Query. Whether you need dashboards for sales, operations, or data monitoring, I can help you present your data effectively.

Data Cleaning (Excel / Python / Power Query)

I clean messy or unstructured data using Excel functions, Power Query, or Python (Pandas). From handling missing values to formatting columns, I ensure your data is clean, consistent, and ready for analysis.

SQL Query Writing

I write optimized SQL queries in MySQL for selecting, filtering, joining, and summarizing data. Whether it's simple reports or complex conditions, I can help you extract the right insights from your database.

Python Data Visualization

I use Matplotlib and Seaborn to create clear and impactful charts like bar graphs, heatmaps, line plots, and more. These visuals help you explore and present your data effectively in reports or dashboards.

Machine Learning (Basic Models)

I build beginner-friendly machine learning models using scikit-learn. I can help you with data preparation, model training, accuracy evaluation, and visual insights — great for academic or prototype-level use.

Excel Automation (Python + AI)

I automate Excel tasks like updating files, formatting data, or generating reports using Python scripts and AI (GPT). Save time by letting me handle repetitive Excel work with smart automation.

Portfolio / Resume Website (HTML & CSS)

I build simple, clean personal websites using HTML and CSS — ideal for portfolios, resumes, or freelancer profiles. I can also help you host it on GitHub Pages for free access online.
